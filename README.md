## 💡 상세 내용

🐱 SeSAC에서 진행한 **팀 프로젝트**입니다. 

고담은 고양이를 담다는 뜻으로, 우리 동네 길냥이들을 컴퓨터 혹은 스마트폰에 담아 간편하게 확인할 수 있는 서비스를 제공합니다. 

사용자들은 길냥이들의 사진과 정보를 업로드하고, 좋아요와 댓글을 남겨 우리 동네 랜드마크로 만들 수 있습니다.

&nbsp;
